# 《恶俗黄油模拟器》设计方案

副标题：今天也在黄文世界努力活着

## 1. 核心定位

《恶俗黄油模拟器》是一个以贵族学院为主舞台的成人向文字 RPG / 模拟器。

玩家是圣铂利亚国际学院的 18+ 成人特招生，家境贫穷、无父无母，靠奖学金和打工支付学费与生活费。玩家隐藏拥有“魅魔体质”，会持续吸引周围人的欲望和注意。学院表面是国际化上流贵族学校，实际运行在极度恶俗的黄文世界规则里：帅哥痴汉、有钱权贵、地下俱乐部、异常设施和公开荒唐事件都属于日常的一部分。

游戏目标不是单一攻略，而是在四年制学院生活中活到毕业。玩家可以保持贞操、努力学习打工、躲避痴汉；也可以主动利用魅魔体质换取资源、进入堕落路线、攻略固定 NPC、成为魅魔女王，或者在各种恶俗事件中收集回想与结局。

## 2. 设计原则

- 玩家永远是主体，关键行为由玩家选择推进。
- 固定 NPC 是长期路线角色，不是一次性随机事件。
- 普通 NPC 可以承担高频恶俗事件，但不抢固定 NPC 的戏。
- 恶俗浓度是同一世界的“浓度调整”，不是三套不同世界观。
- 打工、上课、社交、探索等行动都必须同时影响资源、属性和事件池权重。
- 目标是一次性实现完整系统；UI 可以先朴素可用，但游戏架构不拆半截。

## 3. 世界设定

### 3.1 学院

正式校名：圣铂利亚国际学院。

学院定位：

- 国际贵族学院，四年制。
- 学生、导师、理事、赞助人和校外合作方都默认为成年人。
- 玩家是少数依靠奖学金入学的贫穷特招生。
- 学院拥有学生会、奖学金办公室、图书馆、体育馆、艺术楼、医学研究楼、旧实验楼、地下俱乐部、富人区宿舍和校外打工合作点。
- 旧实验楼地下存在秘密实验室。Extreme 浓度下，人外/异常事件的主要来源是秘密实验室实验品泄漏，而不是切换到另一个奇幻世界。

### 3.2 玩家背景

- 年龄：18+ 成人。
- 身份：特招生。
- 家庭：无父无母。
- 经济：贫穷，需要打工和奖学金。
- 隐藏设定：魅魔体质。
- 目标：支付生活费和学费、维持学业、撑到毕业。

### 3.3 玩家自定义

设置页提供：

- 玩家姓名。
- 性别：女性、男性、非二元、自定义。
- 第二人称称谓。
- 身体设定模板：跟随性别、玩家自定义、模糊处理。
- 文本视角：默认第二人称。
- 露骨文本偏好：暗示、普通成人、详细成人、黑屏跳过。

成人场景生成时必须读取玩家设置，不能默认玩家性别和身体结构。

## 4. 时间与回合

主回合单位：周。

四年结构：

- 4 年。
- 每年 2 学期。
- 每学期 12 周。
- 总计 96 周。

结算节奏：

- 每周：玩家安排行动、触发事件、更新状态。
- 每 4 周：月结算，扣除生活费/房租，检查经济压力。
- 每 12 周：学期结算，考试、奖学金、学业风险、路线阶段推进。
- 第 96 周：毕业结局判定。

每周默认给 6 个行动点，可由疲劳、疾病、特殊状态、NPC 约束或道具改变。

## 5. 玩家属性

### 5.1 生存资源

- 金钱：生活费、学费、道具、脱身手段。
- 学业：奖学金、毕业、退学风险。
- 体力：行动次数、打工效率、事件抵抗。
- 理智：抵抗诱惑、控制魅魔体质。

### 5.2 成人路线属性

- 欲望：魅魔体质副作用，过高会提高事件触发率。
- 羞耻：公开事件、流言和自我认知变化。
- 堕落：主动享乐、地下路线和高浓度事件接受度。
- 贞操：守贞路线和部分结局条件。
- 魅惑：吸引 NPC 和路人痴汉的强度。
- 魅魔泄露：体质暴露程度，影响异常事件、人外事件和研究线。

### 5.3 社会属性

- 名声：学院传闻、固定 NPC 态度、普通 NPC 行为。
- 权贵关注度：越高越容易遇到固定 NPC 和高风险邀约。
- 痴汉遭遇率：普通 NPC 事件密度。
- 自控力：压制欲望、维持守贞线、避免冲动选择。

### 5.4 初始数值

除金钱、债务和计数器外，核心属性范围均为 0-100，所有变化结算后 clamp 到范围内。

### 5.5 基础公式

每周行动点：

- 基础 6 点。
- 体力低于 25：本周行动点 -1。
- 体力高于 85 且理智高于 70：本周行动点 +1。
- 行动点最少 4，最多 7。

月度生活费：

- 每 4 周扣除 900 金钱。
- 金钱不足时，差额转为债务，并额外增加 `ceil(差额 * 0.2)` 的债务利息。
- 当月未付清生活费：理智 -8，名声 -3，权贵关注度 +4。

学期费用：

- 每 12 周结算一次学费差额。
- 学业 >= 80：奖学金优秀，需支付 1000。
- 学业 60-79：奖学金正常，需支付 2000。
- 学业 40-59：奖学金缩水，需支付 3500。
- 学业 < 40：奖学金警告，需支付 5000，并获得 1 层退学警告。
- 退学警告达到 3 层，触发退学类结局前置。

通用事件概率：

```text
痴汉事件权重 =
10
+ 痴汉遭遇率 * 0.35
+ 魅惑 * 0.15
+ 魅魔泄露 * 0.25
+ 恶俗浓度修正
+ 地点风险修正
- 自控力 * 0.10
```

恶俗浓度修正：

- Normal：0。
- Lewd：+10。
- Extreme：+15，并额外启用实验室泄漏权重。

固定 NPC 候选权重：

```text
NPC 候选权重 =
地点匹配 * 3
+ 时间匹配
+ 路线进行中 * 4
+ 近期未出现
- 刚出现过 * 2
+ 玩家欠钱/欠人情 * 3
+ 权贵关注度 * 0.08
+ NPC 执着 * 0.15
+ 随机扰动 0-2
```

## 6. 恶俗浓度

浓度档只改变同一世界里的事件池、文本尺度和异常元素，不改变主设定、固定 NPC、四年目标或结局路线。

### 6.1 Normal：普通恶俗

同一所学院，表面秩序仍像正常贵族学校。

- 公共场所可能出现成人乱象。
- 玩家本人主要遭遇言语骚扰、下流评价、暧昧邀约、权势试探。
- 固定 NPC 更会装，行为更克制。
- 普通 NPC 多承担背景骚扰和短事件。
- 适合守贞、贫穷求生、普通毕业路线。

### 6.2 Lewd：重度黄油

同一所学院，但潜规则更明目张胆。

- 打工地点、社团、学生会、奖学金办公室都可能触发成人事件。
- NPC 的邀约更直接，交易、设局、地下活动更频繁。
- 公共羞耻、特殊校规、体质检测、会所兼职等事件开启。
- 固定 NPC 路线更容易推进到成人情节。
- 适合堕落、赞助人、主动享乐、固定 NPC 深线。

### 6.3 Extreme：魔改人外

同一所学院进一步暴露隐藏异常。人外事件不是换世界观，而是学院地下秘密实验室的实验品、污染样本和失控研究项目跑出来后，渗透进校园与城市日常。

- 在原有权贵/痴汉/学院规则上追加秘密实验室泄漏事件。
- 开启触手藤蔓、兽化实验体、异种样本、梦境污染、异常植物园、旧实验楼、异种俱乐部等事件池。
- 魅魔体质、医学研究线、地下俱乐部、旧实验楼和实验品回收任务权重上升。
- 仍然可以守贞、毕业、堕落、主动享乐或走魅魔女王线。

## 7. 行动系统

每个行动至少影响三类东西：

- 即时收益：金钱、学业、体力恢复、道具。
- 属性代价：疲劳、欲望、羞耻、堕落、名声风险。
- 事件权重：地点事件、NPC 遭遇、固定路线、异常事件。

常规行动：

- 上课：学业上升，体力下降，学院 NPC 权重上升。
- 学习：学业和理智上升，欲望略降。
- 休息：体力和理智恢复，可能触发梦境/邻居事件。
- 打工：赚钱，改变疲劳、名声、NPC 入口和风险。
- 探索校园：地点事件和学院 NPC 遭遇。
- 探索城市：路人痴汉、打工机会、校外固定 NPC。
- 社交：推进固定 NPC，提升权贵关注度。
- 购物：花钱买道具，降低风险或提高诱惑。
- 锻炼：体力和自控力上升，体育线权重上升。
- 主动寻欢：欲望下降，堕落上升，部分路线和回想开启。
- 回避社交：风险下降，但压力和孤立上升。

## 9. 固定 NPC

有名固定 NPC 都是黄油 boss 级角色。普通帅哥、路人痴汉和一次性事件 NPC 不需要详细档案。

固定 NPC 通用动态值：

- 好感。
- 执着。
- 痴汉度。
- 危险度。
- 真心度。
- 关系阶段。
- 最近记忆。
- 路线 flag。
- 本周是否出现。

### 9.1 沈砚洲

- 背景：中国财阀继承人。
- 身份：学生会副会长，掌管奖学金审核。
- 外貌：黑发冷白皮，金丝眼镜，校服扣到最上面。
- 性格：礼貌、控制欲强、擅长用规则逼人。
- 权势：奖学金、处分建议、学生会权限。
- 尺寸设定：22 cm。
- 标签：奖学金、学生会、权力压迫、冷淡痴汉。
- 禁止 OOC：不能写成粗鲁暴力型；他总是礼貌压迫。

### 9.2 白泽礼

- 背景：中日混血，旧贵族家系。
- 身份：学生会长。
- 外貌：银灰短发，浅色眼睛，气质干净到像不能碰。
- 性格：克制、洁癖、说话温柔但危险。
- 权势：学院纪律、学生会最高权限。
- 尺寸设定：21 cm。
- 标签：秩序、纪律室、禁欲崩坏、规则惩罚。

### 9.3 神宫寺莲

- 背景：日本财阀少爷。
- 身份：学院最大赞助人之子。
- 外貌：黑发金瞳，混血感，身高压迫感强。
- 性格：砸钱型变态，喜欢用资源把人圈住。
- 权势：赞助、豪宅、金钱资源。
- 尺寸设定：26 cm。
- 标签：豪宅、包养诱惑、金钱堕落、占有欲。

### 9.4 Adrian Kingsley

- 背景：英裔贵族。
- 身份：交换生，马术社实际赞助人。
- 外貌：金发蓝眼，绅士外表，制服像量身定制。
- 性格：绅士皮、狩猎脑，把追求当贵族游戏。
- 权势：马术社、礼仪课、国际贵族圈。
- 尺寸设定：23 cm。
- 标签：马术社、礼仪课、狩猎感、优雅痴汉。

### 9.5 Lucien Moreau

- 背景：法裔艺术世家。
- 身份：摄影与戏剧社赞助人。
- 外貌：浅棕卷发，绿眼，泪痣，手指漂亮。
- 性格：审美型变态，把玩家当作危险又漂亮的作品。
- 权势：摄影兼职、戏剧社、艺术展。
- 尺寸设定：20 cm。
- 标签：摄影、换装、后台、羞耻名声线。

### 9.6 陆燃

### 9.10 Noah Sterling

- 身份：天才转学生，学院系统权限很高。

### 9.13 固定 NPC 路线阶段

每个固定 NPC 使用统一阶段机，具体事件由 NPC 标签和当前地点决定。

| 阶段 | 名称 | 进入条件 | 作用 |
| ---: | --- | --- | --- |
| 0 | 未接触 | 默认 | 只作为候选背景，不主动推进 |
| 1 | 注意到玩家 | 首次遭遇或对应地点行动 | 开始加入候选池，出现短事件 |
| 2 | 正式盯上 | 好感/执着/危险任一达到 25 | 开启邀约、交易、试探事件 |
| 3 | 路线打开 | 好感或执着达到 45，且触发 2 个个人事件 | 开启个人线、专属地点和记忆 |
| 4 | 路线锁定 | 玩家多次接受邀约，或关键选择偏向该 NPC | 开启成人事件、修罗场和专属结局前置 |
| 5 | 结局前置 | 对应主属性达到阈值，或第 80 周后关系稳定 | 进入 30 结局判定池的强候选 |

NPC 路线推进值：

```text
路线热度 =
好感 * 0.35
+ 执着 * 0.30
+ 真心度 * 0.20
+ 危险度 * 0.15
+ 玩家主动选择次数 * 4
+ 最近 8 周互动次数 * 3
```

阶段推进规则：

- 好感高、真心度高：更容易进入隐藏真爱、赞助人稳定、普通毕业变体。
- 执着高、危险度高：更容易进入权贵锁定、豪门套餐、修罗场。
- 堕落高、玩家主动选择多：更容易进入主动享乐、魅魔女王、地下路线。
- 贞操高且拒绝成功多：更容易进入痴汉败北、守贞类结局。

## 10. 事件系统

事件分为本地模板事件和 API/GM 生成事件。

事件基础字段：

```json
{
  "id": "event_id",
  "title": "事件标题",
  "location": "地点",
  "time": "时间段",
  "density": ["normal", "lewd", "extreme"],
  "npc_type": "none | random | fixed",
  "npc_id": "optional_fixed_npc_id",
  "tags": ["打工", "奖学金", "公共羞耻"],
  "setup": "事件开场",
  "choices": [],
  "state_delta_hint": {},
  "memory_hint": ""
}
```

事件类型：

- 环境目击事件。
- 被搭讪事件。
- 学院传闻事件。
- 打工事件。
- 贫穷压力事件。
- 奖学金事件。
- 固定 NPC 邀约。
- 魅魔体质失控。
- 公开羞耻事件。
- 交易/资助事件。
- 主动寻欢事件。
- 人外/异常事件。
- 秘密实验室泄漏事件。
- 实验品回收/遭遇事件。
- 回想解锁事件。
- 结局前置事件。

### 10.1 事件生产方式

事件内容采用“结构本地固定 + 大批量正文可由 DS/GM 生成”的方式。

笨笨负责：

- 事件 schema。
- 地点、浓度、NPC、属性和结局的权重规则。
- 首批种子事件。
- DS/GM 生成提示词。
- JSON 校验、去重、导入导出。
- 不同事件进入详细描写/黑屏/摘要的流程。

DS/GM 适合负责：

- 批量生成黄油事件正文和变体。
- 按固定 NPC 性格扩写路线文本。
- 按玩家性别、姓名、身体设定改写第二人称叙事。
- 为同一事件生成 Normal/Lewd/Extreme 的演出皮肤。

本地必须能在没有 API 时运行：

- 内置种子事件池保证完整游戏可玩。
- API 开启后，GM 可以替换或补充事件。
- 生成出的事件必须进入本地校验，不能直接污染存档。

### 10.2 事件数量目标

一次性实现目标：

- 本地种子事件不少于 120 个。
- Normal/Lewd/Extreme 每档至少 30 个通用事件皮肤。
- 固定 NPC 每人至少 5 个个人事件，共 60 个。
- 打工事件每个地点至少 5 个，共 40 个。
- 结局前置事件至少 30 个，每个基础结局至少 1 个。
- API/GM 可继续生成并导入更多事件。

### 10.3 选择结构

每个事件至少提供 3-5 个选择，常见选择类型：

- 拒绝/离开。
- 观察/继续看。
- 交涉/讨价还价。
- 接受邀约。
- 利用魅魔体质。
- 主动推进。
- 黑屏跳过。
- 进入详细描写。
- 降低尺度。
- 求助/转移地点。

选择需要显式声明：

- 属性变化。
- 是否进入成人事件。
- 是否推进固定 NPC 记忆。
- 是否解锁回想。
- 是否触发结局前置。

## 11. 成人文本渲染规则

设置中允许成人露骨文本，但文本生成要分层。

### 11.1 固定 NPC 成人事件

固定 NPC 是长期路线角色。玩家明确进入成人事件后，文本可使用：

- 小说式叙述。
- 第二人称指代玩家。
- 根据玩家性别/身体设定调整措辞。
- 保留 NPC 性格和关系阶段。
- 连续推进，不只写几句结算。
- 记录关键结果到 NPC 记忆和回想。

固定 NPC 的详细文本应该体现角色差异，例如控制欲、温柔陷阱、享乐主义、研究欲、赞助关系，而不是所有人同一种语气。

### 11.2 普通 NPC 成人事件

普通 NPC 不是路线重点。即使进入成人事件，也默认：

- 几句话概括。
- 重点写选择后果和状态变化。
- 可以记录成人结果标签。
- 不长篇展开。
- 不抢固定 NPC 路线权重。

### 11.3 黑屏与摘要

玩家可以随时选择：

- 黑屏跳过。
- 摘要结算。
- 降低露骨度。
- 暂停当前事件。

系统应把成人文本能力做成设置项，而不是写死。

## 12. 结局路线

结局由长期属性、路线 flag、固定 NPC 关系和第 96 周状态综合判定。

计划结局共 30 个，结局名偏冷幽默和成就弹窗风格。

### 12.1 毕业/生存类

1. **《真的只是来上学的》**：学业达标，路线未极端化，四年平稳毕业。
2. **《贫穷，但是活着》**：长期缺钱但没有退学，靠打工和省钱撑到毕业。
3. **《奖学金没有被污染》**：学业高、名声稳定、成功保住奖学金。
4. **《孤狼特招生》**：固定 NPC 关系都很低，几乎全程靠自己毕业。
5. **《996 也比这里清纯》**：打工次数极高，成人路线推进低，毕业时疲劳和存款都很突出。

### 12.2 守贞/自控类

6. **《圣铂利亚最后的直立行走者》**：贞操保持，自控高，堕落低。
7. **《痴汉败北合集》**：拒绝次数极高，多个固定 NPC 执着但失败。
8. **《你们学校没有保安吗》**：高痴汉遭遇率下多次成功逃脱或求助。
9. **《魅魔体质，素食主义》**：魅惑高但主动成人事件极低。
10. **《谢谢，不约，赶作业》**：学习/上课权重极高，多次用学业理由跳过邀约。

### 12.3 堕落/主动类

11. **《这世界不对，但来都来了》**：中等堕落，能利用规则生存。
12. **《主线任务：自愿沉沦》**：主动寻欢次数高，欲望管理成功。
13. **《堕落学生，但是绩点还行》**：堕落高但学业没有崩。
14. **《地下俱乐部优秀新员工》**：俱乐部路线深入，Rafael 权重高。
15. **《魅魔女王，堂堂登基》**：魅惑、堕落、主动选择和 NPC 掌控度都高。

### 12.4 权贵/关系类

16. **《赞助人不是奖学金》**：接受长期资助，金钱稳定但自由下降。
17. **《全员单推，但是推你》**：多个固定 NPC 高好感、高执着。
18. **《修罗场也是场》**：多个固定 NPC 冲突，但玩家成功掌控局面。
19. **《金丝雀也是贵族鸟》**：被高权势 NPC 长期锁定，生活条件大幅改善。
20. **《黄文世界居然有纯爱》**：单一固定 NPC 真心度极高，玩家选择稳定。

### 12.5 翻车/坏结局类

21. **《退学通知比情书先到》**：学业过低或学费长期无法支付。
22. **《贫穷是最强 Debuff》**：债务和权贵交易失控。
23. **《校园论坛今日热帖》**：名声风险过高，学院传闻失控。
24. **《欲望值没有上限提示》**：欲望和魅魔泄露长期过高，自控失败。
25. **《你被加入豪门套餐》**：某个高权势 NPC 危险度/执着过高，普通路线被锁死。

### 12.6 异常/掌控类

26. **《体检报告写不下了》**：魅魔泄露、体质检查或特殊关注长期累积过高。
27. **《旧楼层：已读不回》**：频繁探索高风险区域，触发失踪/隔离/锁定路线。
28. **《校规不会写检讨》**：多次触发学院异常规则或特殊校规事件。
29. **《梦里也要交学费吗》**：梦境、压力或体质副作用事件累积过高，现实感开始崩坏。
30. **《最终变量，堂堂上桌》**：玩家反过来掌控异常规则、痴汉生态或高风险事件池，成为本轮最高掌控者。

基础 30 个结局为全浓度共享。恶俗浓度不锁基础结局，只改变达成路径、事件材料和结局演出皮肤。

例如：

- Normal 下，26 号结局可能来自校医体检、学院传闻和普通痴汉关注。
- Lewd 下，26 号结局可能来自高强度体质检测、会所兼职和特殊校规。
- Extreme 下，26 号结局才会替换成秘密实验室、实验品记录和人外样本演出。

后续可以追加浓度专属变体成就，但不计入基础 30 结局。

## 13. 存档结构

存档本地优先。`IndexedDB` 存游戏档案和事件库，`localStorage` 只存当前槽位、UI 偏好和轻量设置。API 只负责生成事件和文本，不拥有唯一存档。

存档规则：

- 支持 1 个自动存档和至少 12 个手动存档槽。
- 每周开始自动存档一次。
- 每次进入高风险事件前自动写入 `pre_event_snapshot`。
- 导出为单个 JSON 文件，包含游戏状态、NPC 记忆、回想、事件库和结局图鉴进度。
- Token、网关 URL、最近模型列表默认不随存档导出，避免把私密配置混进分享档。
- 提供“包含 API 设置导出”的高级开关，默认关闭。
- 导入存档时先校验版本和字段，失败则不覆盖当前档。

每回合使用事务式推进：

1. 生成 `turn_id`，记录当前状态快照。
2. 创建 `pending_turn`，包含玩家选择、地点、候选 NPC、发送给 GM 的摘要。
3. 如果走本地事件池，直接校验事件并应用变化。
4. 如果走 API，收到响应后先 JSON 校验，再生成 `resolved_turn`。
5. 只有 `resolved_turn` 校验通过，才应用属性变化、NPC 记忆、回想和历史。
6. 如果 API 超时或响应坏掉，不推进游戏，只保留可重试的 `pending_turn`。
7. 重试时优先复用同一个 `turn_id`，避免同一周事件被重复结算。

```json
  },
```

## 14. API / GM 接入

游戏必须纯本地可玩，同时设置页支持 API/GM。API 关闭时走本地事件池；API 开启时由 GM 生成或扩写事件，再交给本地规则校验。

### 14.1 设置项

- API 开关：关闭 / 开启。
- 网关 URL。
- Token。
- 模型列表：点击“拉取模型”后从网关读取。
- 当前模型：从模型列表选择，也允许手动输入。
- 请求超时。
- JSON 模式开关。
- 调试日志开关。
- 原始响应保存开关。
- Replay/重试策略开关。

提示词不作为普通设置项。游戏内置 GM 提示词、成人文本提示词、事件 JSON 约束和固定 NPC 写法规则；设置页只暴露必要连接配置。

高级调试区可以提供：

- 查看当前内置提示词。
- 复制当前提示词。
- 临时覆写提示词。
- 恢复默认提示词。

高级覆写只用于调试，不进入普通游玩流程。

### 14.1.1 模型拉取

默认按 OpenAI-compatible 方式拉模型：

```text
GET {gateway_url}/v1/models
Authorization: Bearer {token}
```

成功后读取模型 `id` 列表，写入本地设置缓存。失败时：

- 显示错误。
- 不清空旧模型列表。
- 允许玩家手动填写模型名。
- API 关闭时完全不影响本地游玩。

### 14.2 GM 职责

GM 负责：

- 根据地点、时间、属性和候选 NPC 生成事件。
- 决定固定 NPC 是否登场。
- 给出事件选择和状态变化建议。
- 更新 NPC 动态记忆摘要。
- 管理多 NPC 冲突、邀约和路线推进。

GM 不负责：

- 替玩家选择。
- 无视玩家设置。
- 把固定 NPC 写到 OOC。
- 每次读取全量 NPC 档案。

### 14.3 候选 NPC 裁剪

不要每次把所有 NPC 人设和全部记忆发给 GM。本地先检索和裁剪，每回合只给 0-3 个候选 NPC。

候选权重：

- 当前地点匹配 +3。
- 当前时间匹配 +1。
- 路线进行中 +4。
- 最近没出现 +1。
- 刚出现过 -2。
- 玩家欠钱/欠人情 +3。
- 关注度高 +2。
- 随机扰动 +0-2。

发送给 GM 的 NPC 信息只包含：

- 名字。
- 身份。
- 外貌一句。
- 核心性格。
- 当前关系状态。
- 最近 1-3 条记忆。
- 禁止 OOC 点。
- 与当前事件相关的标签。

```json
{
  "game": "恶俗黄油模拟器",
  "world_rules": [
```

## 15. 一次性实现范围

目标是一次性做完整系统，UI 先保证能玩、能看、能调试，不追求最终视觉完成度。

必须实现：

- 单文件 HTML，可直接本地打开。
- 设置页：玩家姓名、性别、第二人称称谓、身体设定、恶俗浓度、成人文本偏好、API 配置。
- 周回合日历：4 年、8 学期、96 周。
- 每周行动点系统。
- 月度生活费和学期费用结算。
- 属性面板和变化日志。
- 打工系统，包含 8 个打工地点。
- 地点探索，包含学院、城市、地下和旧实验楼相关地点。
- 本地事件池，至少 120 个种子事件。
- API/GM 生成事件，带 JSON 校验和失败回退。
- 固定 NPC 12 人档案、阶段、动态记忆和候选裁剪。
- 普通 NPC 随机事件。
- 成人文本模式：详细、摘要、黑屏、降尺度。
- 回想列表。
- 30 个基础结局和结局图鉴。
- 结局演出皮肤按恶俗浓度变化。
- 存档、读档、导出、导入。
- 事件编辑器：允许用户导入/新增本地事件 JSON。
- 调试面板：查看属性、权重、候选 NPC、最近事件、GM 原始响应。
- 请求重试和 replay 策略，避免 API 超时后重复推进同一回合。

暂不追求：

- 精致 UI。
- 立绘、图片、语音和音效。
- 云同步。
- 多人在线。

## 16. 结局判定优先级

30 个结局会互相冲突，因此必须有判定优先级。

判定顺序：

1. 即时失败类：退学、债务崩坏、强锁定路线。
2. 固定 NPC 强绑定类：单一 NPC route stage 5 且权重极高。
3. 掌控/女王类：玩家主动选择、魅惑、堕落、NPC 掌控度都高。
4. 守贞/自控类：贞操、自控、拒绝成功和低堕落满足条件。
5. 堕落/主动类：堕落、主动寻欢、地下路线满足条件。
6. 权贵/修罗场类：多个 NPC 高执着/高好感。
7. 异常/体质类：魅魔泄露、特殊区域、梦境/体检/实验室权重满足条件。
8. 毕业/普通类：以上都不命中时，按学业、金钱和路线温度选一个。

同一优先级内使用分数最高的结局：

```text
结局分 =
主属性匹配分
+ 相关事件计数 * 3
+ 相关 NPC 阶段 * 6
+ route flag 加成
- 冲突属性惩罚
```

每个结局至少需要：

- 标题。
- 条件函数。
- Normal/Lewd/Extreme 三种演出皮肤或演出差异。
- 解锁说明。
- 回想入口。

## 17. 批量测试与调参

初始数值只要求合理可跑，后续通过批量模拟调整。

批测目标：

- 1000 次随机策略模拟。
- 1000 次守贞优先策略模拟。
- 1000 次赚钱优先策略模拟。
- 1000 次固定 NPC 攻略策略模拟。
- 1000 次堕落/主动路线策略模拟。

需要观察：

- 96 周内退学率是否过高。
- 金钱是否过紧或过松。
- 30 个结局是否都有可达路径。
- Normal 浓度是否也能全结局收集。
- Extreme 是否不会压垮所有普通路线。
- 固定 NPC 是否出现频率均衡。
- 打工收益是否形成有效取舍。
- API/GM 事件是否会破坏本地数值平衡。

初始调参目标：

- 普通随机玩法毕业率约 45%-60%。
- 认真学习打工玩法毕业率约 75%-90%。
- 高风险打工玩法不必然失败，但更容易进入堕落、债务、权贵或地下路线。
- 30 个结局都能通过有意识策略触发。
