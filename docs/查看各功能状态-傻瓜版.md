# 查看网关状态

打开：

```text
你的网关地址/admin/status
```

或使用：

```bash
curl "http://127.0.0.1:5000/admin/status"
```

当前返回区块：

| 字段 | 含义 |
| --- | --- |
| `summary` | 全局摘要是否存在及长度 |
| `r2` | R2 只读是否成功 |
| `dynamic_memory` | 动态记忆数量 |
| `core_cache` | 核心缓存 pending 数量 |
| `notebook` | 当前 R2 记事本数量 |
| `recent_windows` | 最近窗口数量 |
| `last_user_reply` | 本地上次回复时间文件是否存在 |
| `config` | 上游、DeepSeek 摘要和 embedding 配置是否填写 |

某个区块出现 `ok: false` 时看其 `error`，再查对应 systemd 日志。

这个接口不包含聊天白名单、黑名单或 Notion 状态；这些运行能力已经移除。
