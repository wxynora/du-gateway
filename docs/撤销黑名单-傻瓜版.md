# 撤销黑名单（傻瓜版）

误把某个窗口加进黑名单了？按下面做即可解除。

---

## 第一步：先查出「窗口 ID」

在浏览器地址栏输入（把 `localhost:5000` 换成你实际网关地址）：

```text
http://localhost:5000/admin/windows
```

页面会显示一串 JSON。找到 **`"blacklisted": true`** 的那一条，看它前面的 **`"id": "xxx"`**，这个 `xxx` 就是窗口 ID，抄下来备用。

---

## 第二步：发一个「解除黑名单」的请求

任选下面一种方式。

### 方式 A：用 Postman / Apifox（推荐）

1. 打开 **Postman** 或 **Apifox**（没有就搜一下下载，免费）。
2. 新建请求：
   - **方法**选：**DELETE**
   - **URL** 填：`http://localhost:5000/admin/blacklist/你的窗口id`  
     （把 `你的窗口id` 换成第一步抄下来的窗口 ID）
3. 点 **发送**。
4. 返回里出现 `"ok": true` 就表示已经解除黑名单。

### 方式 B：用 curl（命令行）

在终端里执行（把 `你的窗口id` 和 `localhost:5000` 换成实际的）：

```bash
curl -X DELETE "http://localhost:5000/admin/blacklist/你的窗口id"
```

返回里有 `"ok": true` 就表示解除成功。

---

## 总结

| 步骤 | 做什么 |
|------|--------|
| 1 | 浏览器打开 `http://网关地址/admin/windows`，找到 `blacklisted: true` 那条的 **id** |
| 2 | 用 Postman/curl 发 **DELETE** `http://网关地址/admin/blacklist/刚抄的id` |
| 3 | 看到 `"ok": true` 就解除了 |

解除后，该窗口再发消息会按**新窗口**重新判定（不含「测试」会进白名单）。
